# Engram

**Digital traces of Theoretical Neuroscience & NeuroAI.**

This repository consolidates academic papers into persistent memory through **notes** and **code reproductions**.

## Structure

- **Surveys/**: Notes and summaries for survey papers.
- **Implementations/**: Notes and code reproductions for specific articles.

## Usage

Each entry is organized by paper title or topic. Code implementations typically include setup instructions within their respective subdirectories.

> **Note:** All notes are written in [Typora](https://typora.io). When writing in Chinese, mixing Markdown syntax and math formulas often causes rendering issues on GitHub. Despite numerous attempts to fix this, these notes are optimized for local Typora rendering only — GitHub display is not a concern.
