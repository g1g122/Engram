# Engram

**Digital traces of Theoretical Neuroscience & NeuroAI.**

This repository consolidates theoretical foundations, literature notes, and code reproductions for Theoretical Neuroscience and NeuroAI.

## Structure

- **Foundation/**: Foundational notes on mathematics, neuroscience, machine learning, and related concepts.
- **Literature/**: Literature notes, conceptual syntheses, and non-reproduced research articles.
- **Reproduction/**: Code reproductions, experiments, and implementation notes for specific papers.
- **Datasets/**: Local datasets and data files used across the workspace. This directory is ignored by Git.

## Usage

Each entry is organized by paper title or topic. Reproduction projects typically include setup instructions within their respective subdirectories.

> **Note:** All notes are written in [Typora](https://typora.io). When writing in Chinese, mixing Markdown syntax and math formulas often causes rendering issues on GitHub. Despite numerous attempts to fix this, these notes are optimized for local Typora rendering only - GitHub display is not a concern.
